# Win-Win Iterative Model (WWIM)
A lightweight iterative framework for general agents, core ideas:
- Replace static training with continuous iteration
- Use human-AI win-win as value constraint
- Ensure safety and controllability with logical stability

This repository is a basic concept version for academic exchange and prototype validation.

## Structure
- `iter_engine.py`: Basic iteration loop
- `constraint.py`: Simple compliance constraints
- `judge.py`: Basic convergence judgment
- `demo.py`: Simple text conversation example
- `requirements.txt`: Environment dependencies

## Core Logic (Pseudocode)
```python
while not converged:
    state = observe()
    action = choose_action(state)
    reward = win_win_reward(state, action)
    update_model(reward)
